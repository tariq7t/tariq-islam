Extraterrestrial Life Evaluator- Earth Class ELE-E

Personal Project

This is an autonomous object detecting & tracking robot that continuously follows a pre-specified object while remaining at a distance. I implemented object recognition though OpenCV The code is executed by SSH-ing from an external device into the Pi. After object recognition is done, my algorithm measures the area of the object and proceeds to follow it if area is smaller than a specified amount. The object is kept at the center of camera with a simple centroid algorithm. If object moves past center of frame, my robot will turn so object comes back at center.

  • Python
  • OpenCV
  • RaspberryPi
